## API 시작하기
<br/>

<Frame>
    단말기로부터 발생한 매출 데이터와 가맹점 시스템을 연계할 수 있는 부가 서비스를<br/>
    `B2B` API 형태로 제공합니다.

    카드 승인정보(API 제공)는 `여신전문금융업법` 및 `개인정보보호법`에 따라<br/>
    민감 정보에 해당될 수 있어, `금융감독기관`의 검사 대상 사업자에 한해 제공됩니다.<br/><br/>

    따라서 아래 조건을 모두 충족하는 경우에만 서비스 이용이 가능합니다.

    - 금융감독원 검사 대상 법인
    - ISMS 또는 ISMS-P 인증 보유
    - 정보보호 책임자 지정 및 관리체계 보유
    - 유비씨엔과 별도의 API 연동 계약 체결
    <br/>
    ※ 무료로 제공되는 일부 기능의 경우, 변경 사항에 대한 별도의 안내가 제공되지 않을 수 있습니다.
</Frame>
<Frame>
    ### 지원 서비스
    - 거래내역
        - 거래내역 목록(5분 전)
    - 마감정보
        - 특정 일자의 마감된 거래들의 집계정보 목록
    - 상태정보
        - 단말기의 상태정보 (1시간 마다 단말기에서 발송하는 상태정보 제공) 목록

</Frame>
<Frame>
    ### 사전 준비 사항
    유비씨엔 API를 사용하기 위해서는 먼저 가맹점 고유번호(company)와 암호화된 토큰(token)을 발급받아야 합니다. <br/>
    해당 정보는 유비씨엔 관리자에게 문의하여 발급받을 수 있습니다.
</Frame>

    ### API 호출 방법
    API 호출 방법은 다음과 같습니다.
```ts

const API = axios.create({
  baseURL: `http://api.ubcn.co.kr/api`,
  headers: {
    "Access-Control-Allow-Origin": "*",
    "company": "{company}",
    "token": "{token}",
  },
});

// 서비스에 따라 `Method` 가 다를 수 있습니다.
const res = await API.get("{service}");
console.log(res.data);

// 프로세스가 제대로 완료되지 않은 경우 에러 코드가 존재합니다.
if (res.code != null) {
  return alert(res.decription);
}

```
